# PI Van

Aplicativo para gestao de vans escolares com login, cadastro, salas e rotas.

## Estrutura do repositorio

- [pi_van](pi_van) - app Flutter (Android e iOS)
- [docs](docs) - documentacao do projeto

## Setup rapido

1. `cd pi_van`
2. `flutter pub get`
3. `flutter run`

Para configurar Firebase, siga [docs/firebase-setup.md](docs/firebase-setup.md).

## Branch flow

- `main`: producao
- `hml`: integracao
- `sprint_1`: sprint atual
- `sprint_1/feature-login`: branches de task
